每日邮件Action

包含以下内容：
- 每日一句
- 天气信息

## 配置

需要去项目的 `Settings` - `Actions secrets` 中添加以下配置：
- MAIL_HOST：邮件服务器主机
- MAIL_PORT：邮件服务器端口
- MAIL_USERNAME：发件人邮箱用户名
- MAIL_PASSWORD：发件人邮箱密码
- MAIL_ADDRESS：收件人邮箱，多个中间用英文逗号分隔
